    getting started
    welcome to bcm

    The Bitcraft City Manager helps settlement leaders track inventory, plan tier upgrades, manage citizens, and look up game data -- all from one place.

    To get started, enter a Claim ID or search by city name in the header bar and press Enter. Once loaded, the tabs across the top unlock your settlement data.

    BCM is keyboard-first. Every tab has a hotkey -- the underlined letter in each tab label. Press it from anywhere to jump there instantly.

    Press / to jump to the search bar at any time. Press Esc to leave a search box and return to navigation.
    the tabs
    h
    Home -- city overview dashboard with headline stats, active crafts, and material health
    i
    Inventory -- food, supplies, crafting stations, and per-profession breakdowns
    p
    Planner -- calculate what you need for your next tier upgrade
    c
    Citizens -- member roster with skills, gear, activity, and search
    d
    IDs -- look up citizen and item entity IDs for game commands
    m
    Map -- compose bitcraftmap.com links with resource/enemy/player filters
    a
    Calc -- craft time estimator with stamina and food modelling
    r
    Resources -- resource calculator for gathering planning

    Some tabs have their own search -- Citizens, IDs, and Calc all have filter inputs. Press / to focus the search for the current tab.

    the planner

    The planner calculates everything you need to complete a tier upgrade. Select a target tier from the dropdown -- it shows what your settlement has and what's still missing across all five research branches.

    Three views show the same data differently:

    t
    Tasks -- grouped by profession, filterable, copyable to Discord
    f
    Flowchart -- visual dependency flowchart per research branch
    o
    Monitor -- vertical table with polling to observe task progress

    In Tree view, the research tabs along the bottom switch between branches (Stone, Wood, Metal, Leather, Cloth) plus Study Journals. Use Hide done to collapse completed branches.

    The copy buttons export formatted text for Discord. CSV export gives you everything in a spreadsheet-friendly format.
